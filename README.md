A beautifully designed, premium **Chitkara University Semester 4 SGPA & CGPA Calculator**. Built with React, Vite, and Tailwind CSS, this tool empowers students to precisely track their academic performance, featuring dynamic grade estimations, progress visualizations, and a stunning glassmorphism user interface.

## ✨ Features

- **Subject-Specific Grading Logic**: Customized input fields based on subject type (`standard`, `integrated project (IP)`, `environmental sciences (ES)`).
- **Dual Past Semester Modes**: Choose between a *Quick Entry* (overall past CGPA) or *Detailed Entry* (Semester-wise SGPA) to compute your aggregate Cumulative GPA.
- **Dynamic Progress Visualizations**: Stunning animated circular progress rings to visualize SGPA & CGPA metrics instantly.
- **Smart Validation**: Enforces strict input validation to cap marks at their maximum allowed values. Prevents accidental mouse-scroll value changes.
- **Persistent Storage**: Utilizes `localStorage` to securely save your grades locally, ensuring data persists across reloads.
- **Premium Glassmorphism UI**: Uses a responsive, high-end "glass" aesthetic with subtle background orbs and glowing effects for an unmatched user experience.

## 🛠 Tech Stack

- **Framework**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + Custom CSS Variables
- **Animations**: Framer Motion / Motion
- **Icons**: Lucide React

## 🚀 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/raisahibb/Chitkara_Grade_Calculator.git
   cd Chitkara_Grade_Calculator
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:3000` to see the calculator in action.

## ⚙️ Building for Production

To create an optimized production build:
```bash
npm run build
```
You can preview the built application using:
```bash
npm run preview
```
